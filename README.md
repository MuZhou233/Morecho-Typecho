# Morecho - Jekyll

## Get Started

1. clone or fork this repo
1. install ruby on your pc
1. run the following command

```bash
$ gem install bundler jekyll
$ cd /path/to/this/repo
$ bundle install
$ bundle exec jekyll serve
```

you will see your blog [here](http://localhost:4000)
